Getting Started

You will need to make an accountant to accsess this API. For almost all requests are used based and requires a webtoken in to be set in the headers. Click on this Link and then continue.

Base URL

The base URL for librixs is https://librix-server.herokuapp.com/. After this link is where you will add all further calls.

Resources

We have three main routes:

  • /user
  • /books
  • /matches

When we dive deeper into each call it is important to know on post requests what must be there, and what can be added. Everything that is light grey in color is not required to get a correct response.
